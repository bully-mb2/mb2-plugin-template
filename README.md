# MB2 Plugin Template
MB2 Plugin Template is an effort by the [Movie Battles community](https://community.moviebattles.org/) to create templates for the [MB2 Log Reader](https://github.com/bully-mb2/mb2-log-reader). In this project you can find example usage for various languages.
